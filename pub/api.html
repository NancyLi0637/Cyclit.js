Getting Started

A quick start guide for using Cyclit.js.
Note: This library is intended for Vanilla JS.

Setting Up

To use this library, simply include "Cyclit.js" and "Cyclit.css" in your HTML file.
If you don't have jQuery as one of your scripts, include jQuery as well.
In order to access the functionalities in this library, always start by initiating a CyclitGenerator.


Cyclit.js is a library that provides 5 different types of pre-made cycle graphs where each type supports different combinations of information format.
Each CyclitGenerator can create one Card Cycle, one Timeline Cycle, one Frame Cycle, one Cube Cycle, and one Carousel Cycle. The instructions on the usage will be included in the Methods Section.

To test that the library is succesfully included, call the connect() function in your own javascript file and check the correct output is printed in the console.


Basic Usage

There are only two types of methods in this library - one type is to directly create one of the five cycles with given parameters, and another type is methods that relate to each type of cycle.
Both of these two types are extremely easy to use. The first type of methods creates 'JS Objects' that are stored in the CyclitGenerator, and will return the uppermost element which can be directly appended to one of your HTML elements. The second type of methods simply take the element returned by the first type of method and perform DOM manipulation on it.

Create Cycles

These methods directly return an HTML element, which can be appended to your existing element.

.makeCards(images, length, backColor)

Description:Create a Card Cycle with length elements. Each element is flippable upon clicking if enabled.
Requirement: 1<= length <=6
Return Value: A div element containing a Card Cycle

Parameter images length backColor
Type list of String integer color code as String
Description Used as the front face of the cards The number of elements in a cycle Used as the back face of the cards

Description:Create a Timeline Cycle with length elements. Each element is hoverable if enabled.
Requirement: 1<= length

Type list of String integer list of String with limited length list of String
Description Displayed as one container along the timeline The number of elements in a cycle The titles of each cycle element The description of each cycle element

Description:Create a Frame Cycle with length elements. Each element is flippable upon clicking if enabled.
Requirement: 1<= length
Return Value: A div element containing a Frame Cycle

Parameter images length backColor titles contents
Type list of String integer color code as String list of String with limited length list of String

.makeCube(images, length, contents)

Description:Create a Cube Cycle with length elements. The color of the faces are preset and each face is hoverable. The cube is rotatable by calling corresponding functions.
Requirement: 1<= length <=6
Return Value: A div element containing a Cube Cycle

Parameter images length contents
Type list of String integer list of String
Description Used as the hovered face of the cube faces The number of elements in a cycle The description of each cycle element
Usage:

.makeCarousel(images, length, backColor, titles)

Description:Create a Carousel Cycle with length elements. Each element is flippable upon clicking. The carousel can be rotated horizontally by calling corresponding functions.
Requirement: 1<= length <=15
Return Value: A div element containing a Carousel Cycle

Parameter images length backColor titles (optional)
Type list of String integer color code as String list of String/list of Integer index
Description Used as the front face of the cells The number of elements in a cycle Used as the back face of the cells The titles of each cycle element
